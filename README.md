# Tractogram Transformation (app-tract-transform)

This repository provides a reproducible workflow to apply spatial transformations to **tractograms (`.tck`)** using:

```bash
applyTransformations2tract.sh
```

The pipeline supports transformations generated by:

* **ANTs**
* **FSL (FLIRT/FNIRT)**

The app is designed for **Brainlife.io** and runs inside a **containerized environment** to ensure reproducibility and portability.

---

# Features

This pipeline performs:

* application of **affine and nonlinear transformations** to tractograms
* support for **single tractogram input** or **folders containing multiple `.tck` files**
* compatibility with **ANTs transforms**
* optional conversion of **FSL transforms to ANTs format**
* optional affine conversion to improve compatibility
* optional inverse transformation
* optional reverse operation for warp/affine order
* tractogram transformation using **scilpy**

All processing is executed inside a **containerized environment** to guarantee reproducibility across systems.

# Field descriptions

## Tractogram input

Exactly **one** of the following must be provided:

| Field         | Description                             |
| ------------- | --------------------------------------- |
| `input_tract` | Input tractogram (`.tck`)               |
| `tcks`        | Folder containing multiple `.tck` files |

If both are provided, the app exits with an error.

---

## Moving-space image input

The app selects the **first available** non-null field among:

| Field | Description                    |
| ----- | ------------------------------ |
| `t1`  | Moving-space T1-weighted image |
| `t2`  | Moving-space T2-weighted image |
| `fa`  | Moving-space FA image          |

---

## Fixed/reference image input

The app selects the **first available** non-null field among:

| Field      | Description                       |
| ---------- | --------------------------------- |
| `t1_fixed` | Fixed/reference T1-weighted image |
| `t2_fixed` | Fixed/reference T2-weighted image |
| `fa_fixed` | Fixed/reference FA image          |

---

## Transform fields

| Field          | Description                              |
| -------------- | ---------------------------------------- |
| `affine`       | Linear transformation (`.mat` or `.txt`) |
| `warp`         | Forward nonlinear deformation field      |
| `inverse-warp` | Inverse nonlinear deformation field      |

---

## Optional parameters

| Field               | Description                    |
| ------------------- | ------------------------------ |
| `invert`            | Apply inverse linear transform |
| `reverse_operation` | Apply warp before affine       |
| `verbose`           | Enable verbose output          |

### Warp selection logic

The app selects the deformation field as follows:

1. if `reverse_operation = true`, use `warp`
2. else if `invert = true`, use `inverse-warp`
3. otherwise, use `warp`

# Output

The output of the app is the transformed tractogram or transformed set of tractograms.

## Single input tractogram

```bash
transformed/<basename_of_input_tractogram>
```

Example:

```bash
transformed/track.tck
```

## Folder input (`tcks`)

Each input `.tck` file is transformed independently and saved to:

```bash
transformed/<basename_of_each_input_file>
```

# Usage on Brainlife.io

This app is designed to run on **Brainlife.io**.

## Web UI

1. Locate the **app-tract-transform** app
2. Select either:

   * a single tractogram, or
   * a folder containing `.tck` tractograms
3. Select the moving and fixed anatomical images
4. Select the affine and deformation fields
5. Execute the pipeline

## CLI

```bash
bl login

bl app run --id <app_id> \
           --project <project_id> \
           --input input_tract:<tract_object> \
           --input t1:<moving_t1_object> \
           --input t1_fixed:<fixed_t1_object> \
           --input affine:<affine_object> \
           --input warp:<warp_object> \
           --input inverse-warp:<inverse_warp_object>
```

# Usage locally

## 1. Clone the repository

```bash
git clone https://github.com/gamorosino/app-tract-transform.git
cd app-tract-transform
```

## 2. Prepare configuration

Edit `config.json` to specify:

* one tractogram or a folder of tractograms
* moving image
* fixed/reference image
* affine
* warp / inverse-warp
* optional flags

## 3. Run the pipeline

```bash
./main
```

This will:

* read the configuration file
* select the appropriate anatomical inputs
* select the appropriate deformation field
* run `applyTransformations2tract.sh` inside the container
* save the transformed tractogram(s) to `transformed/`
