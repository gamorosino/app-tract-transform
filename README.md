# Tractogram Transformation (app-tract-transform)

This repository provides a reproducible workflow to apply spatial transformations to **tractograms (.tck)** using the script:

```
applyTransformations2tract.sh
```

The pipeline supports transformations generated by:

* **ANTs**
* **FSL (FLIRT/FNIRT)**

The app is designed for **Brainlife.io** and runs inside a containerized environment to ensure **reproducibility and portability**.

---

# Features

This pipeline performs:

* Application of **affine and nonlinear transformations** to tractograms
* Compatibility with **ANTs transforms**
* Optional conversion of **FSL transforms to ANTs format**
* Optional affine composition
* Optional inverse transformation
* Robust tractogram transformation using **scilpy**

All processing is executed inside a **containerized environment** to guarantee reproducibility across systems.

# Inputs

All parameters are provided via a `config.json` file.

### Example `config.json`

```json
{
  "tractogram": "input/tract.tck",
  "reference": "input/reference.nii.gz",
  "affine": "input/0GenericAffine.mat",
  "warp": "input/1InverseWarp.nii.gz",
  "invert": true
}
```

---

# Field descriptions

| Field        | Description                                                         |
| ------------ | ------------------------------------------------------------------- |
| `tractogram` | Input tractogram (`.tck`)                                           |
| `reference`  | Reference image defining the target space                           |
| `affine`     | Linear transformation (ANTs `.mat` or converted FSL transform)      |
| `warp`       | Nonlinear deformation field (`Warp.nii.gz` or `InverseWarp.nii.gz`) |
| `invert`     | Apply inverse affine transform                                      |

Optional parameters may include:

| Field               | Description                                                |
| ------------------- | ---------------------------------------------------------- |
| `pre_affine`        | Optional pre-affine transformation                         |
| `compose`           | Affine composition mode                                    |
| `convert_affine`    | Convert affine to ANTs affine type                         |
| `reverse_operation` | Apply warp before affine (required in some ANTs workflows) |

# Output

The output of the app is the **transformed tractogram**.

Output location:

```
transformed/<basename_of_input_tractogram>
```

Example:

```
transformed/tract_transformed.tck
```

# Usage

### 1. Clone the repository

```bash
git clone https://github.com/gamorosino/app-tract-transform.git
cd app-tract-transform
```

---

### 2. Prepare configuration

Edit the `config.json` file to point to your tractogram and transformations.

---

### 3. Run the pipeline

```bash
./main
```

This will:

* read the configuration file
* run `applyTransformations2tract.sh`
* apply the specified transformations to the tractogram
* save the output tractogram to `transformed/`
